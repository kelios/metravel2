- **Остров слёз** ~7, **Верхний город** ~4, **Парк Горького** ~3, **Большой театр** ~1, **Зыбицкая/Немига** ~2, **Комаровка** ~4 — единичные, вперемешку с личными; смотреть глазами.

## Рекомендация по обложке и галерее статьи

- **Обложка** (cover, `collection=travelMainImage`): кадр из `2016/Стары Минск` (зимний старый город с подсветкой) или яркое цветение из `2014/Лошицкий парк`/`Батанический сад`.
- **Галерея** (`collection=gallery`): по 3–5 лучших из Лошицы 2014, Ботсада 2014, Стары Минск 2016, плюс отобранные одиночные (Остров слёз, Большой театр, Комаровка).
- Точкам маршрута (`coordsMeTravel`) при добавлении в Мастере можно подставлять фото из этих папок — EXIF подставит координаты.

## Как заливать
Через Мастер травела (web): шаг «Медиа» — обложка + галерея (`POST /upload`), шаг «Маршрут» — точки с фото. Мастер сам делает slug, превью и модерацию. Это безопаснее скрипта.
