```

```

**Нормальное состояние:**
```
[Отмена]  [👁️ Превью]  [📄 Сохранить PDF]
```

**При загрузке:**
```
[Отмена]  [⏳ Превью]  [⏳ Создание...]
            ↑              ↑
         disabled       disabled
         opacity: 0.5   + крутится
```

**При ошибках валидации:**
```
[Отмена]  [👁️ Превью]  [📄 Сохранить PDF]
            ↑              ↑
         disabled       disabled
         opacity: 0.5   opacity: 0.5
```

## Анимации

### Spinner при загрузке
```
Frame 1:  ⏳  (0°)
Frame 2:  ⏳  (90°)
Frame 3:  ⏳  (180°)
Frame 4:  ⏳  (270°)

animation: spin 1s linear infinite
```

### Открытие модалки
```
Frame 1: opacity: 0, scale: 0.95
Frame 2: opacity: 0.5, scale: 0.97
Frame 3: opacity: 1, scale: 1

transition: all 0.3s ease
```

### Hover на кнопке
```
Frame 1: y: 0, shadow: light
Frame 2: y: -1px, shadow: medium

transition: all 0.2s cubic-bezier(0.4, 0, 0.2, 1)
```

```css
box-shadow: 0 4px 6px -1px rgba(0, 0, 0, 0.1),
            0 2px 4px -1px rgba(0, 0, 0, 0.06);

box-shadow: 0 10px 15px -3px rgba(0, 0, 0, 0.1),
```
