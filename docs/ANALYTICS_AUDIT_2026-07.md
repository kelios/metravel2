## 2. Диагноз

**Что хорошо**
1. SEO-машина работает: клики +42% за месяц, позиция 8.2, статьи стабильно собирают трафик, крошки валидны, рост ускоряется с середины июня.
2. Поведение читателей здоровое: 12 мин, глубина 7, отказы 10%.
3. Технически чисто: 0 критичных ошибок разметки.

**Что плохо (по убыванию критичности)**

1. **Квесты невидимы.** В Google за 3 месяца: **0 кликов, 2 показа** (warsaw-syrenka, grodno-royal). Внутренних заходов — 16 юзеров/мес (в основном свои). Причины, подтверждённые проверкой прод-HTML:
   - в SSG-HTML статей **нет ни одной ссылки на `/quests`** (квест-промо рендерится только клиентом) → для Google квесты — сироты, для читателя статики блок может вообще не появляться до гидратации;
   - у квест-страниц **нет `meta description`** и og-разметки (только title + скрытый h1) → «crawled, not indexed»;
   - в sitemap квесты есть, но без ссылочного веса из статей этого мало.

2. **Регистрация мертва, потому что до неё никто не доходит и незачем.** `/registration` — 6 просмотров (2 человека) за месяц; «отправка формы» = 0. В HTML статьи (90% входного трафика) нет ни CTA, ни причины регистрироваться: читатель получил ответ про озеро — и ушёл. Ценность аккаунта (избранное, свои маршруты, прогресс квестов, PDF-книга) на странице статьи не предъявляется никак.

3. **Воронка не измерима.** Используется 1 из 200 целей (wizard_step_view). Нет целей: регистрация, логин, старт/шаг/финиш квеста, добавление в избранное, создание маршрута. «Никто не участвует» сейчас — ощущение, а не метрика.

4. **Соцсети как канал отсутствуют** (3 визита/мес) — весь трафик на одной игле SEO.

5. **SEO-хвост:** 284 crawled-not-indexed; CTR 2.6% не растёт — страницы с большими показами и низким CTR: закшувек (30/3107 = 1%), energylandia (65/4015 = 1.6%), польские мальдивы (23/1518), смолевуд (17/1084).

6. Расхождение GSC (~25–40 кликов/день) vs Метрика (~14 поисковых визитов/день) — известное (consent + адблок ловит ~15%), учитывать при чтении цифр; фактический трафик выше видимого в Метрике в ~2–3 раза.

## 3. План

### Фаза 0 — фундамент измерений (FE, 1–2 дня) — сначала, иначе эффект фаз 1–2 невидим
- [ ] Цели в Метрике + `reachGoal` на фронте: `registration_complete`, `login_success`, `quest_start`, `quest_point_done`, `quest_finish`, `favorite_add`, `travel_publish`, `cta_register_click`.
- [ ] Проверить, что счётчик Метрики стоит на всех SSG-страницах и стреляет из SPA-навигации (по расхождению с GSC).
- [ ] Дашборд «активация»: визиты → cta_click → registration → первое действие.

### Фаза 1 — квесты: дать вход и снять барьер (FE + контент, ~неделя)
- [ ] **Квест-блок в SSG-пререндер статей** соответствующего города (сервер-рендеренная ссылка+картинка, не client-only). Приоритет статей: краковские (пляжи 133 клика, закшувек, энергиландия) → краковские квесты; минские → минские.
- [ ] **SEO-мета квест-страниц**: meta description, og:title/description/image, JSON-LD (TouristAttraction/HowTo), видимый текст-лендинг (что это, сколько точек, длительность, старт) в пререндере.
- [ ] `/quests` — полноценный индексируемый лендинг-хаб по городам в SSG.
- [ ] **Гостевой режим квеста**: первые 1–2 точки без регистрации, дальше — «сохраним прогресс» → мягкая регистрация. Сейчас барьер на входе убивает конверсию при 16 посетителях/мес.
- [ ] Прогнать квест-URL через ручную индексацию GSC (~10/день) + IndexNow.
- Метрика успеха: ≥15 квест-страниц с показами в GSC, ≥30 `quest_start`/мес.

### Фаза 2 — регистрация: причина + путь (FE, ~неделя)
- [ ] Инлайн-CTA в статье (в SSG): после контента и у карты — «Сохрани маршрут / свои места» → login-wall с внятным value.
- [ ] Гостевое ♥/«в поездку» с deferred-регистрацией: действие разрешаем, при сохранении просим аккаунт (сейчас гость, вероятно, вообще не видит смысла).
- [ ] Одно-кликовый вход (Google OAuth) на login/registration; минимум полей.
- [ ] Автоцель «заполнил контактные данные» = 2 при «отправил» = 0 — проверить форму регистрации на фрикцию/ошибки отправки (e2e-прогон формы на проде).
- Метрика успеха: ≥20 регистраций/мес к ревью 08.08.

### Фаза 3 — SEO-хвост (продолжение текущей рутины)
- [ ] CTR-оптимизация топ-20 «показы высокие, CTR <2%» (закшувек, energylandia, смолевуд, польские мальдивы): title с интентом + эмоцией, meta description с ценой/сезоном/как добраться.
- [ ] Разобрать 284 crawled-not-indexed + 31 redirect-error + 3×403 (index-doctor; 403 — возможно бот-защита, отдать тикетом на бэк, если серверное).
- [ ] Продолжать seo-daily (ручная индексация 10/день, IndexNow).

### Фаза 4 — второй канал (owner, параллельно)
- [ ] Соцсети: короткие Reels/TikTok по топ-локациям со ссылкой на статью+квест (контент уже есть — фото статей). Pinterest для фото-статей.
- [ ] В шаблон статьи — заметные share-кнопки (2 перехода в соцсети/мес — им нечем делиться).
