Новая задача

Строка допишется в таблицу docs/AGENT_WORKBOARD.md. ID присвоится автоматически.

Не удалось прочитать мост автоматизации напрямую из файла. Браузеры блокируют fetch() локальных файлов при открытии через file://. Запусти статический сервер из корня репозитория и открой борд по http: npx serve . → http://localhost:3000/docs/AGENT_WORKBOARD_LOCAL.html (или python3 -m http.server). Данные обновляет npm run workboard:heartbeat.
