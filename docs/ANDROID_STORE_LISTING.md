## Полное описание (≤ 4000 символов)

```
MeTravel — это путеводитель по реальным путешествиям. Внутри — рассказы о поездках с точками на карте, городские квесты-прогулки и идеи, куда поехать на выходные.

ЧТО ВНУТРИ

• Путешествия с картой. Каждый маршрут — это история с фотографиями, координатами мест и практическими деталями: как добраться, что посмотреть, сколько времени заложить.

• Интерактивная карта. Все точки маршрутов на одной карте — стройте свой план и смотрите, что есть рядом.

• Городские квесты. Пешие маршруты-головоломки по городам: на каждой точке — живая история места и задание, которое нужно выполнить, стоя перед объектом. Отличный способ узнать город, а не просто пройти мимо.

• Поиск мест. Находите достопримечательности, природные локации и интересные места по названию и на карте.

• Избранное. Сохраняйте маршруты и места, чтобы вернуться к ним перед поездкой.

• Идеи для поездок. Подборки направлений по Беларуси и не только — для коротких вылазок и больших путешествий.

ДЛЯ КОГО

Для тех, кто любит путешествовать самостоятельно: без турагентств, по своему плану и в своём темпе. Беларусь, ближнее зарубежье и популярные направления — с проверенной практической информацией от реальных путешественников.

Сайт проекта: metravel.by

Приложение бесплатное. Часть ссылок на бронирование отелей и туров — партнёрские.
```

## Data Safety evidence matrix

Текущие Android permissions в `app.json`:

- `ACCESS_COARSE_LOCATION`, `ACCESS_FINE_LOCATION`;
- `CAMERA`, `READ_MEDIA_IMAGES`, `READ_MEDIA_VIDEO`;
- `RECORD_AUDIO` явно заблокирован.

Permission не равен автоматически «данные собираются»: для ответа Play Console
нужно сверить runtime flow, server persistence, retention и sharing.

| Data family | Фактический Android flow | Что проверить перед декларацией |
|---|---|---|
| Location | foreground location используется для карты/точек рядом; quest geofencing может запросить background permission в соответствующем flow | передаётся ли location backend/external provider, retention, optionality |
| Photos/media | image picker/camera используются для avatar, article/travel/quest media | какие файлы загружаются на server/CDN, удаление и sharing |
| Account/email | auth/profile backend-dependent functions | required/optional, account deletion и retention |
| Push token/device identifier | `expo-notifications` запрашивается по необходимости | token storage, purpose, retention и provider role |
| Analytics | native analytics path сейчас намеренно disabled; web GA4/Yandex загружаются только после consent | не переносить web declaration в Android автоматически; проверить собранный artifact |

- **Шифрование при передаче:** да (HTTPS).
- Account/data deletion path проверяется в установленной сборке и production API;
  только после этого его можно обещать в Play Console.
- Ответ «shared with third parties» определяется правилами Play для фактических
  processors/providers в момент заполнения; этот draft не даёт юридического ответа.
