## Сравнение цветовых палитр

| Элемент | До (❌) | После (✅) | Разница |

## Философия цветовой палитры

### Старая (яркая)
```
Характеристики:
• Яркие неоновые цвета
• Холодные синие оттенки
• Высокая насыщенность (93%)
• Глянцевый вид
• Визуально агрессивная

Использование:
❌ Устаревший тренд 2020-2021
❌ Режет глаза в темноте
❌ Плохо для длительного чтения
```

### Новая (матовая)
```
Характеристики:
• Мягкие матовые цвета
• Теплые натуральные оттенки
• Умеренная насыщенность (23%)
• Матовый вид
• Визуально спокойная

Использование:
✅ Современный минималистичный дизайн
✅ Приятна для глаз
✅ Подходит для долгого чтения
✅ Профессиональный вид
```
