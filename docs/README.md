# MeTravel2
 
 MeTravel2 — кроссплатформенное приложение (Expo / React Native) с web-версией (Expo Router).
 
 ## Структура репозитория
 
 - `app/` — entry и роутинг (Expo Router)
 - `components/` — UI компоненты
 - `src/` — бизнес-логика/сервисы
 - `utils/` — утилиты
 - `__tests__/` — тесты
 
 ## Быстрый старт
 
 ```bash
 npm install
 cp .env.dev .env
 npm start
 ```
 
 ## Запуск платформ
 
 ```bash
 npm run web
 npm run ios
 npm run android
 ```
 
 ## Тесты
 
 ```bash
 npm run test
 npm run test:coverage
 ```
