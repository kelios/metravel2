## 💻 Примеры кода

### Базовое использование
```typescript
import { getThemeConfig } from '@/services/pdf-export/themes/PdfThemeConfig';

// Получить тему
const theme = getThemeConfig('black-white');
```

### В генераторе PDF
```typescript
const pdf = await generator.generate(travels, {
  theme: 'newspaper',
  format: 'A4',
  orientation: 'portrait',
});
```

### Проверка доступности
```typescript
import { PDF_THEMES } from '@/services/pdf-export/themes/PdfThemeConfig';

// Все доступные темы
const allThemes = Object.keys(PDF_THEMES);
// ['minimal', 'light', 'dark', ..., 'black-white', 'sepia', 'newspaper']
```

## 🚀 Быстрый старт

1. **Импортируй тему**
   ```typescript
   import { getThemeConfig } from '@/services/pdf-export/themes/PdfThemeConfig';
   ```

2. **Выбери подходящую**
   - Официально? → `black-white`
   - Ностальгия? → `sepia`
   - Динамика? → `newspaper`

3. **Примени**
   ```typescript
   const pdf = await generator.generate(travels, { theme: 'sepia' });
   ```
