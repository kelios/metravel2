## 🚀 Как использовать

### Для пользователя:

1. **Открыть настройки фотоальбома**
   ```
   Выбрать путешествия → Настройки → Тема оформления
   ```

2. **Выбрать газетную тему**
   ```
   Раскрыть группу "📰 Газетная верстка (NEW!)"
   Выбрать одну из трех опций
   ```

3. **Прочитать описание**
   ```
   Под select появится подсказка с описанием темы
   ```

4. **Сгенерировать PDF**
   ```
   Нажать "Сохранить PDF" → Получить газетную верстку
   ```
