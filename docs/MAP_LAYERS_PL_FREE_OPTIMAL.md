# Оптимальный бесплатный слой мест ночёвки (вариант A)

Выбранный вариант: **OSM (OpenStreetMap) + Overpass API**.

Это не «ведомственный» слой LP, но:
- бесплатный
- легальный (ODbL при корректном attribution)
- устойчивый (много зеркал Overpass)
- даёт реальные POI: camp_site / camp_pitch / shelter / wilderness_hut

## Где работает
Слой **не ограничен Польшей**: он показывает точки в любой стране, где они есть в OSM.

## Русификация
- Название слоя и типы объектов в попапах показаны по‑русски.
- Если в OSM заполнен `name:ru`, он будет использоваться в первую очередь.

## Как включить
Слой подключается как web‑оверлей и подтягивает точки для текущего viewport.

Можно сменить endpoint Overpass через env:
- `EXPO_PUBLIC_OVERPASS_ENDPOINT` (по умолчанию `https://overpass-api.de/api/interpreter`)

Attribution:
- `© OpenStreetMap contributors (ODbL)`

## Ограничения
Чтобы не перегружать Overpass:
- запрос делается с дебаунсом
- есть лимит на площадь bbox (если сильно отдалиться — слой временно пустой)
