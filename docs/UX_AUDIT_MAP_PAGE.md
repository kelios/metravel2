## Общая информация

### Цель страницы
Интерактивная карта для поиска достопримечательностей, маршрутов и интересных мест в Беларуси и соседних регионах.

### Целевая аудитория
- Путешественники по Беларуси и соседним странам
- Любители автопутешествий и велотуризма
- Искатели необычных мест (заброшки, природные объекты)

### Основной сценарий
1. Пользователь открывает карту → видит точки рядом с текущим местоположением
2. Фильтрует по категориям (церкви, руины, родники и т.д.)
3. Выбирает интересное место → строит маршрут или сохраняет

## Детальные проблемы

### Критические (High)

#### 1. Onboarding блокирует UI
- **Описание:** Tooltip перехватывает pointer events, клик вне tooltip вызывает навигацию на /quests
- **Влияние:** Пользователь случайно уходит со страницы
- **Файл:** `components/MapPage/MapOnboarding.tsx`

#### 2. Нет текстового поиска
- **Описание:** Пользователь не может найти конкретное место по названию
- **Влияние:** Вынужден скроллить карту вручную

#### 3. Карта центрируется на Кракове
- **Описание:** При отсутствии геолокации карта показывает Польшу вместо Беларуси
- **Влияние:** Дезориентация, непонимание контекста
- **Файл:** `constants/mapConfig.ts`

### Средние (Medium)

#### 4. Слишком длинные названия мест
- **Описание:** Названия не обрезаются, выходят за границы карточек
- **Файлы:** `components/MapPage/AddressListItem.tsx`, `components/MapPage/MapPeekPreview.tsx`

#### 5. Непонятные cluster numbers
- **Описание:** Числа на карте без контекста (что это — количество мест?)

#### 6. Badge "99+"
- **Описание:** При 221 месте показывает "99+" — потеря информации
- **Файл:** `screens/tabs/MapScreen.tsx`

### Низкие (Low)

#### 7. Радиус 500-600 км бесполезен
- **Описание:** Слишком большой диапазон, покрывает всю страну
- **Файл:** `components/MapPage/FiltersPanelBody.tsx`
