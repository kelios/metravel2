  цель перелинковки + первоисточник фактов. Кандидат уверенный.

**Создан квест** `oshmyany-crossroads` — «Ошмяны: город четырёх флагов»
- id 44, city id 40, **7 точек** + intro + финал. Сквозная нить: маленький город,
  где веками уживались 4 государства и 3 веры.
- Маршрут (запад→восток, ~2 км, пеший): Синагога (1912) → Краеведческий музей
  им. Богушевича (опц., часы работы) → Костёл Св. Михаила («две белые свечи») →
  Церковь Воскресения → Мельница/электростанция на Ошмянке → Руины костёла
  францисканцев → Кресты на горе Кальвария (финал).
- Факты сверены (статья 252 + ru.wikipedia/planetabelarus/poshyk/shtetlroutes);
  задания только по стабильным наблюдаемым деталям. Скорректировано ресёрчем:
  костёл БЕЛЫЙ оштукатуренный (не краснокирпичный), мельница сейчас магазин
  (не кафе), кресты на Кальварии считать нельзя (число плавает) → эти вопросы
  сделаны свободными `any_text`.
- **Гео-сверка** (`quest-geocheck`): все 7 точек reverse-geocode-верифицированы —
  каждая стоит ровно на своём объекте в OSM (Сінагога 122Б, музей=былая аптека 128,
  Касцёл Міхаіла, Царква Уваскрэшання, Электрастанцыя, Касцёл францысканцаў,
  Крыжы на Кальварыі). Координаты — авторские GPS из статьи 252.
- **Обложка** (cover.png, PIL-иллюстрация: две белые башни + холм крестов + река)
